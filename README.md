WebTool
=======

a website to provide many developing, daily tool.

## this project uses
* HTML, Js, CSS, PHP
* [SASS](http://sass-lang.com/)
* [Riot](http://riotjs.com/)
* [npm](https://www.npmjs.com/)
* [PM2](http://pm2.keymetrics.io/)

## set up dev environment
```
npm i; ./UTL/precompile.js;
```

## run dev server
```
npm run dev;
```
